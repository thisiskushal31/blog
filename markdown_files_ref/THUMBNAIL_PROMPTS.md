# Thumbnail Generation Prompts for ChatGPT/DALL-E

## Image Specifications
- **Dimensions**: 1536 x 1024 pixels (3:2 aspect ratio) - exact size required, other similar sizes acceptable
- **Format**: PNG
- **Style**: Simple, clean, professional technical blog cover image
- **Theme**: Database deployment guides - should convey technical expertise and comprehensive coverage
- **Important**: 
  - **NO TEXT** - purely visual icons and logos only, no text labels, no words, no letters (except logos that are inherently text-based like "AWS")
  - No overlapping icons, no duplicate icons, clear spacing between all elements

## Usage Instructions

1. Copy each prompt to ChatGPT (GPT-4 with DALL-E) or another image generation tool
2. Request the image in 1536x1024 pixel dimensions
3. Save the generated images as PNG files
4. Name them exactly as:
   - `postgresql-deployment-guide.png`
   - `mysql-deployment-guide.png`
   - `mongodb-deployment-guide.png`
   - `redis-deployment-guide.png`
   - `aerospike-deployment-guide.png`
   - `database-mastery-index.png`
   - `elasticsearch-deployment-guide.png` (for updating existing thumbnail)
   - `database-comparison.png` (for relational vs NoSQL guide)
5. Place all images in: `blog/public/blogImages/`
